Troubleshooting

Bookmarklet doesn't run

  • Make sure you're on an https:// site (won't work on local files)
  • Check that GitHub Pages is enabled and your files are accessible
  • Verify you replaced YOUR-USERNAME in the bookmarklet code

No rules appear

  • Open browser console (F12) and check for errors
  • Verify inject.js has your correct GitHub username in RULES_URL
  • Ensure rules.json is valid JSON (use jsonlint.com)

Rules match wrong text

  • The script uses word boundaries (\b) to prevent partial matches
  • If common words like "Power" or "Blast" are matching too broadly, consider making rule names more specific

Technical Details

How It Works

  1. Bookmarklet injects inject.js into the current page
  2. Script fetches rules.json from GitHub Pages
  3. Rules are cached in localStorage for 24 hours
  4. DOM is scanned for text matching rule names
  5. Matching text is wrapped with interactive elements
  6. MutationObserver watches for new content and processes it automatically
